# Project Management Application #


## Project Installation ##

    npm install
    npm install -g gulp
    npm install -g json-server

## Creating Assets | Looks the changes for the files ## 

    npm run build && npm run watch

## Sass through Gulp ##
    
    gulp && gulp watch

## Deploying the frontend app ##

    npm run start

## Fake Rest Api Server ##

    npm run json-server
